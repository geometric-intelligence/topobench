# DHGNN Challenge Submission Draft

  This folder tracks the draft Track 2 submission for DHGNN, based on:

  Ma et al., "Directed Hypergraph Representation Learning for Link Prediction," AISTATS 2024.

  Paper: https://proceedings.mlr.press/v238/ma24b/ma24b.pdf

  Official implementation: https://github.com/mazitong/DHGNN

  This is an exploratory draft. The first implementation step is to verify how the original link-prediction model maps to
  the TopoBench/GraphUniverse challenge pipeline.
